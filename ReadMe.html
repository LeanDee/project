Live in the Moment

This is a website which presents yoga classes, and also it gives at first information regarding the yoga and it positive effects on a human body.

Design

I have chose the  purple color as the main color for the pages background.

Features

Navigation bar has been placed on all pages and it’s fixed. The website includes four pages which are : Home, Yoga, Gallery and the last one is About. It has a black color font which stands out nice in white background.

The banner

The banner is there to attract the attention of the user and also it provides some information regarding the courses being held weekly.

Website Info

The main page offers information regarding yoga and the history of it as well as the programs that they offer to the users.

The Shortcut section

It is there to help the user for better navigation experience and provide as well information such as contact data and as well accessibility to the appointment booking portal.

The Footer

It allows the user to find us on other social media platforms and also information regarding the copyright policy.
